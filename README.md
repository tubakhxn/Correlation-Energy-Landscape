# Correlation Energy Landscape

## Overview
This project visualizes cross-asset correlation as an evolving energy landscape in 3D. It uses synthetic data to simulate how correlations between assets change over time, representing them as a morphing surface with scientific and neon visual style.

## Features
- Pure Python implementation
- Animated 3D visualization using matplotlib (mplot3d + FuncAnimation)
- Black background, neon purple/violet accents, and smooth semi-transparent surface
- No web UI, no Streamlit/Dash
- Professional scientific look

## Files
- `main.py`: The main Python script. Contains all code for generating data, plotting, and animating the landscape.

## How to Run
1. Make sure you have Python 3.7+ installed.
2. Install required packages:
   ```bash
   pip install matplotlib numpy
   ```
3. Run the project:
   ```bash
   python main.py
   ```
   This will open a window with the animated 3D correlation energy landscape.
